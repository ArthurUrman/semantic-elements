What are semantic elements?

Semantics is defined as the study of meaning in language according to vocabulary.com. Elements in HTML are the commands that are used to define the structure of a webpage. Semantic elements in HTML means that the element's description (name) has a meaning which is easy to understand both for the browser and the developer. See the table below for some examples of semantic elements.


  • Figure - used to add title or explanation to an image
  • Article - specifies a seperate content within the webpage
  • Header - specifies a tile or a header for document or a section within the webpage
